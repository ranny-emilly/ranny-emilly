## A little about me
Gosto de programacao que exige raciocinio de verdade: resolver problema, pensar em eficiencia, entender estrutura por baixo e escrever solucao com cuidado.

Hoje, o que mais aparece por aqui e programacao competitiva, estudo de algoritmos e implementacoes em C e C++, especialmente em estruturas de dados, ponteiros, memoria e logica.

Tambem tenho interesse forte em backend e desenvolvimento de sistemas, principalmente quando o projeto envolve modelagem, CRUD, autenticacao e organizacao de fluxo de dados.

## Featured project
### SIDCC - Sistema Integrado de Dados do Cancer Cervical

- Backend em Go
- Frontend em HTML, CSS e JavaScript
- Banco de dados PostgreSQL
- Projeto academico com proposta real para sistemas de saude, desenvolvido para organizar cadastro de pacientes, historico clinico e fluxo de fichas citopatologicas.
- Cadastro de pacientes, busca, autenticacao, recuperacao de senha e organizacao de dados clinicos

Repositorio: [SIDCC](https://github.com/ranny-emilly/SIDCC-Sistema_Integrado_de_Dados_do_Cancer_Cervical)

Deploy: [projeto-integrador-bo2d.onrender.com](https://projeto-integrador-bo2d.onrender.com/)
